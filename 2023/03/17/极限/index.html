求极限

方法： 先定型后定法，定法之前先四化

定型

未定式：化简 -> 定法

已定式：带入求极限

化简
  1. 非0因子可淡化
  2. 加减法中存在可拆化
  3. “$\sqrt{x}$”有理化
  4. 幂指函数幂指转换
定法
  1. 等价无穷小代换（x->0时）
  2. 泰勒公式
  3. 洛必达法则
  4. 极限四则运算

7种未定式（题型）

$\frac{0}{0} $ 型

  1. 等价无穷小代换（x->0时）
  2. 泰勒公式
  3. 洛必达法则
  4. 极限四则运算

$\frac{\infty}{\infty} $ 型

  1. 抓大头
  2. 上下同除最大项
  3. 洛必达法则

$0 * \infty $

  1. $\frac{0}{\frac{1}{\infty}} = \frac{0}{0} $
  2. $\frac{\infty}{\frac{1}{\infty}} = \frac{\infty}{\infty} $

$\infty - \infty$

  1. 通分
  2. 使$x=\frac{1}{t} $

$1^{\infty}$

==$\lim f(x)^{g(x)} = e^{\lim g(x)\ln f(x)-1}$==

$0^{\infty} $和$0^{0}$

幂指转换
